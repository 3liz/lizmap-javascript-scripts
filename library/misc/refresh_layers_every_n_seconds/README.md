## Description

Refresh layers every N seconds.

Note: 

* [Use this script for Lizmap >= 3.8](https://github.com/3liz/lizmap-javascript-scripts/blob/master/library/misc/refresh_layers_every_n_seconds/refresh_layers_every_n_seconds_3.9.js)
* [Use this script for Lizmap < 3.7](https://github.com/3liz/lizmap-javascript-scripts/blob/3b20f9302b746239b2ff4e85c8f684961ae55842/library/misc/refresh_layers_every_n_seconds/refresh_layers_every_n_seconds.js)

## Configuration

Edit the script to configure the variables

* `QGIS_LAYER_NAMES` containing the list of **QGIS** layers name
* `REFRESH_INTERVAL_MS` with the interval in milliseconds
